Community-level chemical metrics for exploring genomic adaptation to environments (2023)

This vignette runs the code to make the plots from the following paper:

Dick JM, Kang X. 2023. chem16S: community-level chemical metrics for exploring genomic adaptation to environments. Bioinformatics 39(9): btad564. doi: 10.1093/bioinformatics/btad564

This vignette was compiled on 2024-02-22 with JMDplots 1.2.19 and chem16S 1.0.0-11.

library(JMDplots)

Data sources: The GlobalPatterns dataset is from the phyloseq package (McMurdie and Holmes, 2013) and is adapted from Caporaso et al. (2011). The mouse gut dataset was created by analyzing sequence data from Schloss et al. (2012) (downloaded here and used in the mothur MiSeq SOP and DADA2 Pipeline Tutorial) with the DADA2 pipeline modified to use the GTDB training set (Alishum, 2022) for taxonomic classifications.
