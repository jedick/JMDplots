Yeast Gene Expression in Hyper- and Hypo-osmotic Shock

This vignette from the R package JMDplots version 1.2.17 shows chemical metrics for proteins coded by genes that are differentially expressed in hyperosmotic and hypoosmotic shock in yeast. The analysis is described in more detail in a paper (Dick, 2021). Abbreviations:

  • ZC – average oxidation state of carbon.
  • nH2O – stoichiometric hydration state per residue (Dick et al., 2020) computed using the QEC basis species (glutamine, glutamic acid, cysteine, H2O, O2).
datasets <- pdat_yeast_stress(2020)

Differences are calculated as (median value for proteins coded by up-regulated genes) - (median value for proteins coded by down-regulated genes). Circles and squares stand for hyperosmotic and hypoosmotic experiments, respectively.

In the table, values of ΔZC, ΔnH2O, and ΔGRAVY are multiplied by 1000, values of ΔpI and ΔMW are multiplied by 100, and negative values are shown in bold. Abbreviations:

  • ndown – number of down-regulated genes; nup – number of up-regulated genes.
  • pI – isoelectric point; GRAVY – grand average of hydropathicity.
  • nAA – protein length; MW – molecular weight per residue.
